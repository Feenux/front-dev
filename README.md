# Weather-Forecast
The app uses:

- AngularJS 1.5.x
- Angular Material 1.1.x

## Installation and usage

Install npm, install yarn and run with start:
```bash
$ yarn install
$ yarn start
```

Generating build:
```bash
$ yarn build
```
